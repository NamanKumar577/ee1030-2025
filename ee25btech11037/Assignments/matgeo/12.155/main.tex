\textbf{Question: }
The eigenvalues of a matrix are $i$, $-2i$ and $3i$. The matrix is
\begin{multicols}{4}
    \begin{enumerate}
        \item unitary
        \item anti-unitary
        \item Hermitian
        \item anti-Hermitian
    \end{enumerate}
\end{multicols}
\vspace{2mm}


\textbf{Solution:}\\
We know that all the eigenvalues of an anti-Hermitian matrix are either $0$ or purely imaginary.\\
Therefore, the matrix is anit-Hermitian.\\
Thus option $4)$ us correct
